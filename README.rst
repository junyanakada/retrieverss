###########
RetrieveRSS
###########

A feed reader as web application

********
Features
********

* Read entries
  * View entries in all feeds
  * View entries in each feeds
  * View an entry
  * Manage reading status(Read or Unread)
* Manage feeds
  * Add the feed
  * Remove the feed
  * Import the OPML
  * Export the OPML
* Aggregate feeds
  * Periodically
  * Manually
* Manage users
  * Sign in
  * Sign out
  * Create the user
  * Delete the user
  * Reset password
